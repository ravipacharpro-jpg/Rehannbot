# FREE STUFF PROMO BOT

Telegram bot that gives videos after users join required groups.

## Deploy on Render.com

1. Create bot from @BotFather
2. Get your Telegram ID
3. Add environment variables:
   - BOT_TOKEN = your_token
   - OWNER_ID = your_telegram_id
4. Deploy!

## Commands

**Admin:**
- /setgroups @group1 @group2 - Set groups
- /addvideo - Add new video
- /videos - List all videos
- /stats - Bot statistics
